# AGM commands reference

- `agm {br|branch} sync`: `brsync.sh`
- `agm config {cp|copy}`: `cponfig.sh`
- `agm {wt|worktree} {co|checkout}`: `mkwt.sh`
- `agm {wt|worktree} new`: `mkwt.sh -b`, alias for `agm wt co -b`
- `agm {wt|worktree} {rm|remove}`: `rmwt.sh`
- `agm dep`: `pm-dep.sh`
- `agm fetch`: `pm-fetch.sh`
- `agm init`: `pm-init.sh`
- `agm open`: `pm.sh open`
- `agm new`: `pm.sh new`
- `agm {co|checkout}`: `pm.sh co`
- `agm run`: `sandbox.sh`
- `agm tmux new`: `tmux.sh`
- `agm tmux layout`: `tmux-apply-layout.sh`
